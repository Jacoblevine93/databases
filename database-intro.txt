- 42

A: String, number, date, float, money

2. Explain when a database would be used. Explain when a text file would be used.

A: Whenever you need to store a large amount of data. You'd use a text file if you only needed to store a small amount of data.

3. Describe one difference between SQL and other programming languages.

A: SQL is declarative, most other programming languages are imperative or functional.

4. In your own words, explain how the pieces of a database system fit together at a high level.

A: Data is sent to the database or server in a compressed format and then retrieved when needed by the site or application that has the ability to call the database. 

5. Explain the meaning of table, row, column, and value.

A: A table stores all the data given. The column stores a specific type of data for record. The row stores the entire record. And the value is whatever is in each individual cell, which is a combination of both the rows and columns selected. 

6. List three data types that can be used in a table.

8. Given this users table, write SQL queries using the following criteria and include the output:

- The email and sign-up date for the user named DeAndre Data.
